Current Work
MSc Dissertation: Robustness of 3D CNN Alzheimer's Classification to Age Distribution Shift

Developing deep learning pipelines in PyTorch to classify Alzheimer's disease from structural MRI, focusing on fairness across age groups and robustness to distribution shift. Using interpretability methods to diagnose whether models exploit age-correlated shortcuts versus disease-specific pathology.
