**Leaky Integrate-and-Fire Neuron Model**

Python implementation of the leaky integrate-and-fire (LIF) neuron model, a fundamental computational neuroscience model of neuronal spiking behavior based on RC circuit dynamics.
Simulates membrane potential dynamics with threshold-and-reset mechanism. Includes comprehensive analysis: frequency-current curves, inter-spike interval distributions, rheobase calculation, and multi-level current responses demonstrating realistic cortical neuron behavior.

Tech: Python, NumPy, matplotlib

Concepts: Computational neuroscience, neuronal dynamics, RC circuits, spiking models, F-I curves



**Excitatory-Inhibitory Neural Network**

Two-population rate model based on the Wilson-Cowan framework, simulating the dynamics of interacting excitatory and inhibitory neural populations with real-time visualization.
Demonstrates transient responses, E/I balance, and inhibition-stabilized dynamics. Features animated network visualization with activity-dependent coloring, population-level firing rate traces, and pulsed input protocols showing millisecond-scale realistic neural dynamics.

Tech: Python, NumPy, matplotlib, NetworkX

Concepts: Population dynamics, Wilson-Cowan model, E/I balance, rate models, network visualization
